Under the hood

SMILES/FASTA validation, canonicalisation, multi-drug parsing, property tier-resolution.

02
Surrogate ML Layer

57 XGBoost / GNN / QSAR models — each predicts properties + uncertainty for one principle.

What You're Probably Wondering

Principal Investigators

Is this just an LLM wrapper?

No. Zero LLMs in the prediction path. Every score is a mathematical model — DEs, ML regressors, ab-initio physics. LLMs only narrate reports.

Computational Pharmacists

What if a property isn't anywhere?

4-tier resolver computes from first principles — QM, group-contribution, atomistic. Output carries provenance (T1/T2/T3/T4).

R&D Directors
